RadSim runtime prompt reference

The live system prompt is not maintained in this file.

Runtime source:
- Base prompt: radsim/prompts.py, RADSIM_SYSTEM_PROMPT
- Tool-use prompt fragment: radsim/prompt_fragments/tool_use.md
- Personality prompt fragment: radsim/prompt_fragments/personality.md
- Dynamic composition: radsim/prompts.py, get_system_prompt()
- Prompt size breakdown: run /prompt-stats inside RadSim

This file exists only to prevent a stale second copy of the prompt from
drifting away from the code RadSim actually runs.
